This is a restaurant db that has tables that are interrelated. Here tables are built in drizzle orm and has authorization.

The API allows you to utilize the CRUD functionalities

CRUD Operations: Create: Add new records to the database, such as restaurants, menus, menu items, orders, customers, employees, reviews, and reservations.
Read: Retrieve existing records from the database, either individually or as collections.
Update: Modify existing records to reflect changes, such as updating menu items, order statuses, or customer information.
Delete: Remove records from the database, such as deleting old reservations or removing discontinued menu items.
